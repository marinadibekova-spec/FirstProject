Visual Studio Code (VS Code)

A lightweight, extensible code editor from Microsoft — powerful for editing, debugging, and extending.

What is VS Code?

VS Code is a free, open-source (MIT) source-code editor that runs on macOS, Windows, and Linux. It combines a fast editor, built-in Git support, debugging, and a rich ecosystem of extensions.

Key features

  • IntelliSense: smart completions and signature help for many languages.
  • Debugging: integrated debugger with breakpoints, call stacks, and an interactive console.
  • Extensions: install thousands of extensions to add languages, themes, and tools.
  • Version control: built-in Git UI for commits, branches, and diffing.
  • Customization: tweak keybindings, settings, and layouts to fit your workflow.

Tips

  • Open the command palette with Cmd + Shift + P (macOS) or Ctrl + Shift + P (Windows/Linux).
  • Quickly toggle the terminal with Ctrl + `.
  • Use extensions like Prettier for formatting and ESLint for linting JavaScript/TypeScript.
  • Open a folder to get full workspace features and debugging configurations.

Quick-start

To start editing:

  1. Install VS Code from the official website.
  2. Open a project folder: File → Open Folder...
  3. Install recommended extensions for your stack via the left sidebar.
